Browser Security

  • Securing your web browser can give you a safer experience for browsing the web. Web browsers like Microsoft Edge, Mozilla Firefox, and Apple Safari are the default browsers when you bought a new computer. These web browsers are not usually set up in a secure default configuration. If the user does not secure their preferred web browser, many problems can occur from malware to spyware being installed without the user’s permission.

  • There are many factors that you might have threats on your computer:

    • Many users click on links without considering if they are sketchy or not
    • URL addresses can take you to somewhere else
    • Many software packages may include additional software that may increase the chances of vulnerability
    • Third-party software may not receive any security updates
    • Many websites require the user’s permission to allow certain features or install additional software
    • Users do not know how to secure their browser
    • Users are not willing to enable or disable functionality to help secure their preferred web browser.

    Exploiting vulnerabilities became very easy for attackers to user’s computer systems.

  • Many web browsers have different ways of securing your web browser. Each browser listed has a website about their documentation about securing your preferred web browser.
