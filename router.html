Router Security

  • Note: This is generalize and the steps may vary from router to router.

    • From a computer connected by ethernet (your computer is connected directly to your router via Ethernet cable), open your preferred web browser and type in "192.168.0.1" in the Address Bar and hit enter.
    • Click on Wireless Setup
    • To your left, click on the "Multiple SSID" link.
    • Click the drop-down in step ONE and select a name from the list separate from the main network name, (this should be in the box)
    • Enable the SSID Broadcast
    • Set to Broadcast the SSID
    • Either use the default SSID Name or create your own in all capital letters.
    • Set the Separate Subnet to Enable.
    • Enter the following:

      • DHCP Start Address: 10.10.50.2
      • DHCP End Address: 10.10.50.254
      • SSID Subnet Mask: 255.255.255.0
      • Gateway Address: 10.10.50.1
      • Click "APPLY"
